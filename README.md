## SSH keys

1. Generate new key: `ssh-keygen`
2. Add keygen to relevant sites (e.g. Github, Bitbucket, etc)

## Postgres setup

1. `brew upgrade postgresql`
2. `brew postgresql-upgrade-database`
3. `sudo pip install pipenv`
